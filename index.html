게임 소개

1인칭 시점의 퍼즐 어드벤처 게임입니다.

레이저와 거울을 활용한 독특한 퍼즐을 풀며 미스터리한 이야기를 풀어나가세요.

숨겨진 노트를 수집하고, 키패드 암호를 풀며, 제한 시간 안에 퍼즐을 완성하세요.

주요 특징

  • 레이저/거울 퍼즐 - 물리 기반 광학 퍼즐 메커니즘
  • 노트 시스템 - 발견한 단서와 메모를 기록
  • 키패드 입력 - 문자 기반 암호 해독 시스템
  • 다양한 스테이지 - 10개 이상의 독특한 레벨
  • 타임 챌린지 - 제한 시간 내 퍼즐 해결
  • 인벤토리 시스템 - 아이템 수집 및 사용

데모 다운로드

⚠️ 현재 데모 버전을 준비 중입니다.

곧 다운로드 링크가 업데이트될 예정입니다!

Windows 다운로드 (준비 중)

시스템 요구사항

최소 사양

  • OS: Windows 10 64-bit
  • CPU: Intel i5 / AMD Ryzen 5
  • RAM: 8GB
  • GPU: GTX 1060 / RX 580
  • 저장공간: 6GB
  • DirectX: Version 11

권장 사양

  • OS: Windows 10/11 64-bit
  • CPU: Intel i7 / AMD Ryzen 7
  • RAM: 16GB
  • GPU: GTX 1080 / RX 5700
  • 저장공간: 6GB
  • DirectX: Version 12

설치 방법

  1. ZIP 파일 다운로드
  2. 원하는 위치에 압축 해제
  3. HamoniaDemo.exe 실행

문제 해결

  • 게임이 실행되지 않으면 DirectX 최신 버전을 설치하세요
  • 그래픽 드라이버를 최신으로 업데이트하세요
  • 바이러스 백신이 차단하는 경우 예외 처리하세요
